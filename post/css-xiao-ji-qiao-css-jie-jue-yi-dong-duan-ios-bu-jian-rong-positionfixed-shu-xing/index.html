CSS-小技巧css解决移动端ios不兼容position:fixed属性

2019-04-28

移动端开发仿app头部底部固定设置position:fixed，android2.2以上已经实现。但是在ios8以下系统，当小键盘激活时，都会出现位置浮动问题 如何解决：

只需要在中间部分外层div添加css样式,就可以实现效果

position:fixed;
top:50px;
bottom:50px;
overflow:scroll;

demo

.head{top:0;}

</style>
